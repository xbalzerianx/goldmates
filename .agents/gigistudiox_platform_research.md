## Platform Deep Comparison

### 1. 🟡 ETSY (You're already here)
| | |
|---|---|
| **Fees** | $0.20 listing + 6.5% transaction + 3% + $0.25 payment processing = **~12–15% total per sale** |
| **Audience** | 90M+ active buyers. Massive built-in search traffic |
| **Best for** | Templates, mockups, printables, branding kits |
| **Pros** | Huge organic traffic, buyers already looking for design assets, strong SEO |
| **Cons** | High fees, lots of competition, algorithm changes hurt visibility |
| **Verdict** | Keep it. It's your discovery engine. |

---

### 2. 🟢 CREATIVE MARKET ⭐ HIGH PRIORITY
| | |
|---|---|
| **Fees** | **50% commission** — you keep 50% of every sale |
| **Audience** | Designers, agencies, businesses buying premium assets |
| **Best for** | Fonts, templates, graphics, branding kits, UI kits |
| **Pros** | Higher-quality buyers willing to pay more, professional audience, no listing fees, auto-payouts at $20 threshold |
| **Cons** | Need to apply & get approved, $12 minimum price per product, 50% cut |
| **How to apply** | creativemarket.com/sell — need a portfolio, easier if you already have Etsy sales |
| **Real seller data** | One designer made $1,498 in revenue (~$1,048 after cut) with just 17 products |
| **Verdict** | ✅ APPLY NOW. Different, higher-spending audience than Etsy. |

---

### 3. 🟢 CREATIVE FABRICA ⭐ HIGH PRIORITY
| | |
|---|---|
| **Fees** | **50% if they refer the buyer, 75% if YOU refer the buyer** |
| **Audience** | Crafters, designers, print-on-demand sellers — subscription-based buyers |
| **Best for** | Fonts, SVGs, templates, clipart, craft files |
| **Pros** | Subscribers download a LOT — recurring exposure, 75% cut on self-referred sales |
| **Cons** | $120 minimum payout threshold via bank transfer, some seller complaints about IP disputes |
| **How to apply** | creativefabrica.com/open-store |
| **Verdict** | ✅ Good for volume. Their subscribers download constantly. |

---

### 4. 🟢 PAYHIP ⭐ BEST OWN STOREFRONT
| | |
|---|---|
| **Fees** | Free plan: **5% per sale** / Plus $29/mo: 2% / Pro $99/mo: **0%** |
| **Audience** | YOUR audience — people you send there directly |
| **Best for** | Building your own brand store, selling bundles, email list building |
| **Pros** | Built-in affiliate program, coupon codes, email marketing, Apple Pay, 100+ payment methods, EU VAT handled |
| **Cons** | No built-in marketplace traffic — you bring the buyers |
| **Verdict** | ✅ Set this up as your HOME BASE. Link from your Etsy bio, social media, everywhere. Free to start. |

## Recommended Strategy for GigiStudiox

### Phase 1 — Do NOW (Free)
1. ✅ Keep Etsy active — maintain listings, optimize SEO
2. 🆕 **Apply to Creative Market** — higher-value buyers, professional audience
3. 🆕 **Set up Payhip store** — your own branded storefront, 0 upfront cost, 5% fee

### Phase 2 — Within 1 month
4. 🆕 **Apply to Creative Fabrica** — tap into their subscription base
5. 🔗 **Cross-link everything** — Etsy bio → Payhip store, Payhip → "also on Etsy/CM"
6. 📦 **Create bundles** — bundle 5–10 designs together, sell only on Payhip (higher margin)

### Phase 3 — Scale
7. Start building an **email list** via Payhip (free feature)
8. Use **affiliate program** on Payhip to get others promoting your designs
9. Consider Sellfy when you're doing consistent monthly volume
